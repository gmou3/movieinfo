# movieinfo
Get movie info in the terminal
![Screenshot](screenshot.png)

# Dependencies
asciiart  (for default mode) <br />
catimg  (for realistic [-r] mode) <br />
 —  (for no image [-n] mode)

# Usage
./movieinfo.sh [flags]

flags: <br />
   -r: realistic image using catimg (instead of default asciiart) <br />
   -n: no image
